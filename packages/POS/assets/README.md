# Icon Placeholder

This directory should contain application icons in various formats:

- `icon.png` - PNG icon (512x512 or larger)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon

You can use tools like:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [electron-builder icon generation](https://www.electron.build/icons)

To generate icons from a single PNG file.

For now, the application will use default Electron icons until custom icons are provided.
